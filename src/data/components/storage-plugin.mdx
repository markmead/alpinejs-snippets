---
title: How to save to local storage
group: Storage
---

Based on the Alpine JS `$persist` plugin, I create the [persist extended](https://alpinejs.dev/plugins/persist) plugin that includes `$persistGet` and `$persistDelete` magic methods for managing data saved in local storage.

<Callout>
  Make some changes and refresh the page to see your changes saved.
</Callout>
